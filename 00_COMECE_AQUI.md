## 💾 Como Salvar/Compartilhar

### Como HTML (Para Visualizar no Browser)
```bash
# Opção 1: Usar Markdown viewer online
# Copiar conteúdo do arquivo .md
# Colar em: https://markdown-it.github.io/

# Opção 2: Converter com Pandoc
pip install pandoc
pandoc ARQUITETURA.md -o ARQUITETURA.html

# Opção 3: Usar VSCode com extensão
# Instalar: Markdown Preview Enhanced
```

### Como PDF (Para Imprimir)
```bash
# Opção 1: VSCode
# Abrir arquivo .md → Clique no PDF

# Opção 2: Pandoc
pip install pandoc
pandoc ARQUITETURA.md -o ARQUITETURA.pdf

# Opção 3: Browser
# Abrir em browser → Ctrl+P → Salvar como PDF
```

### Como Compartilhar
```bash
# Opção 1: GitHub (Recomendado)
git add *.md
```
